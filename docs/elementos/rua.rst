.. _Rua:

Rua 🜃
=====

A Rua em um nível imediato é a :ref:`IncautasPág`, o contato das Incautas com o :ref:`Blouco`. Conforme a Incauta se envolve, ou quando alguém convida diretamente uma pessoa para acessar uma página das outras :ref:`Divindades`, têm acesso a outras páginas que também compõem a Rua,

De forma mais abstrata, a Rua representa toda a Web, uma vez que a entrada para o Blouco pode vir de qualquer uma de suas páginas, e também nossos arredores físicos ao interagirmos com o Blouco em nossos dispositivos.
